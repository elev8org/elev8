Explore Scratch! (Grades 1–4)

This program introduces elementary students to coding through fun, hands-on lessons using Scratch. Kids explore basic computer science concepts while building games, animations, and interactive stories.

Quick Facts

  • Student Target Age: Grades 1–4
  • Class Size: 8–10 kids (1:4 teaching ratio, minimum 2 staff)
  • Schedule: Weekly, 75 min per session
  • Days: Monday, Tuesday, or Friday (3:45–5:00 PM)
  • Location: Kerrisdale Community Centre

Lesson Plan

Throughout this program, students will gradually learn coding concepts such as sequencing, loops, events, conditionals, variables, and basic problem-solving skills while building interactive stories, games, and animations.

Lesson Topic Concepts Learned
1 Dance Party Sprite movement, sequencing
2 Maze Starter Arrow key input, basic conditionals
3 Hide and Seek Events, timers, interactions
4 Pong Starter Variables (score), motion, loops
5 Catch Game Collision detection, loops
6 Pen Flower Loops, creative patterns, coordinates
7 Stamp Studio Events, sprite costumes, creative control
8 Space Invader Conditionals, loops, game logic
9 Story 1 Sequencing, events, storytelling
10 Story 2 Story continuation, music, backdrop changes
