Explore Scratch! (Grades 1–4)

Explore Scratch! is a coding program where high school student volunteers guide elementary school students through fun, hands-on lessons in computer science using Scratch and introductory HTML website design.

The program blends creativity with logic, encouraging kids to build their own games, animations, and interactive stories while learning how computers think and follow instructions.

Quick Facts

  • Student Target Age: Grades 1–4
  • Class Size: 8–10 kids (1:4 teaching ratio, minimum 2 teaching staff)
  • Schedule: Weekly, 75 min (6 sessions)
  • Days: Monday, Tuesday, or Friday (3:45–5:00 PM)
  • Format: Step-by-step lessons, interactive demos, student project time

What Kids Will Learn

  • Use Scratch to build fun projects (games, animations, stories)
  • Understand key coding concepts: loops, events, sequences, conditionals, variables
  • Develop creativity, logic, and problem-solving skills
  • Get a light introduction to HTML and website structure
  • Gain confidence in digital literacy and computational thinking

Materials & Tools

  • Laptops or desktop computers with internet access (1 per student preferred)
  • Projector (for lesson presentations)
  • Tables and chairs for group learning

Why This Program?

This program bridges the digital literacy gap by:

  • Introducing coding at an early age
  • Promoting youth leadership
  • Making technology fun and approachable
  • Creating a peer-supported learning environment
