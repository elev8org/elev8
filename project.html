Explore Scratch! (Grades 1–4)

Explore Scratch! is a coding program where high school student volunteers guide elementary students through fun, hands-on lessons in computer science using Scratch and introductory HTML.

Students will learn logic, creativity, problem-solving, and basic coding concepts. Each lesson introduces a new programming concept or technique.

Lesson Plan Overview

Lesson Activity New Concepts Taught
1 Dance Party Basic sprite movement, using blocks to control motion
2 Maze Starter Arrow key controls, collision detection, scoring basics
3 Hide and Seek Adding sprites, timers, and event responses
4 Pong Starter Ball physics, scoring system, simple animation
5 Catch Game Custom sprite interactions, increasing difficulty
6 Pen Flower Loops, angles, and creating patterns
7 Stamp Studio Costume changes, new effects, user interactions
8 Space Invader Game customization, simple enemy AI, scoring
9 Story 1 Interactive storytelling, adding dialogue and backdrops
10 Story 2 Continuing story, adding music and animations
