Explore Scratch! (Grades 1–4)

Explore Scratch! is a coding program where high school volunteers guide elementary students through fun, hands-on lessons in computer science using Scratch and introductory HTML website design.

The program blends creativity with logic, encouraging kids to build their own games, animations, and interactive stories while learning how computers think.

Quick Facts

  • Student Target Age: Grades 1–4
  • Class Size: 8–10 kids (1:4 teaching ratio, minimum 2 teaching staff)
  • Schedule: Weekly, 75 min (10 sessions)
  • Days: Monday, Tuesday, or Friday (3:45–5:00 PM)
  • Location: Kerrisdale Community Centre
